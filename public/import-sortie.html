Importer une sortie

Upload GPX + PDF Strava en une étape. Le PDF Strava (cue-sheet) est lu côté client (OCR si nécessaire) et chaque direction est projetée sur le tracé GPX pour créer des POIs.

1

Fichier GPX

Glisse le GPX (export Strava ou ailleurs). Le tracé sera analysé pour distance, D+, points kilométriques.

2

PDF Strava (optionnel)

Si tu as exporté la route en PDF depuis Strava, dépose-le ici. Les directions turn-by-turn seront extraites et placées comme POIs sur le tracé. Si le PDF est scanné (pas de texte natif), OCR Tesseract.js prend le relais (lent : ~10s/page).

Glisser un PDF Strava ici ou clique pour parcourir
3

Métadonnées

Titre, date, chapitre. Si le GPX contient un nom dans <trk><name>, il sera suggéré.
