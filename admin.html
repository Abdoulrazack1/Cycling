Points d'intérêt

Tous les POIs créés par les utilisateurs ou par les admins, toutes sorties confondues. Pour ajouter un POI, ouvrez la page d'une sortie et utilisez le formulaire dédié.

Auto-import de courses

Ce module scrape automatiquement les sources publiques (Miles Republic, FFC HdF) et liste les courses détectées. Pour chaque course, vous pouvez générer le tracé GPX, le profil altimétrique et les POIs en un clic.

Génération manuelle

Pour générer une course depuis ses waypoints (lat/lng) : un GPX et le profil altimétrique seront produits automatiquement (OSRM cycling + Open-Meteo).

              Outils d'administration
            

            
Actions de masse pour gérer le club : envoyer un mail groupé, basculer en mode maintenance, modifier plusieurs comptes simultanément.

          


          
          

            

              
Broadcast email

              SMTP requis
            

            
Envoie un message à un groupe de membres en un clic. Le contenu est wrappé dans un template HTML branded (en-tête CCS + signature footer). Tape un texte simple — les retours à la ligne deviennent des paragraphes.


            

              

                
                
              

              

                
                
                Une ligne vide entre 2 blocs = nouveau paragraphe. Max 10 000 caractères.
              

              

                
                
              

              

              

                
                
              

            

          


          
          

            

              
Mode maintenance

              Effet immédiat
            

            
Quand activé, les visiteurs non-admins reçoivent 503 sur toute écriture (POST/PUT/PATCH/DELETE). Les lectures (GET) restent ouvertes, les admins continuent à passer normalement. Idéal pour une opération courte sur la BDD.


            

              
              Vérification de l'état actuel…
            


            

              

                
                
                Affiché dans la réponse 503. Garde-le court et factuel.
              

              

                
                
              

            

          


          
          

            

              
Actions groupées sur les comptes

              Multi-sélection
            

            
Coche plusieurs membres puis applique une action commune : désactivation en masse (fin de saison), réactivation, ou changement de rôle. Ton propre compte est exclu automatiquement pour éviter de te bloquer.
