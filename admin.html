Points d'intérêt

Tous les POIs créés par les utilisateurs ou par les admins, toutes sorties confondues. Pour ajouter un POI, ouvrez la page d'une sortie et utilisez le formulaire dédié.

Auto-import de courses

Ce module scrape automatiquement les sources publiques (Miles Republic, FFC HdF) et liste les courses détectées. Pour chaque course, vous pouvez générer le tracé GPX, le profil altimétrique et les POIs en un clic.

Génération manuelle

Pour générer une course depuis ses waypoints (lat/lng) : un GPX et le profil altimétrique seront produits automatiquement (OSRM cycling + Open-Meteo).

Journal d'audit
