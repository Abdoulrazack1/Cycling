Mon Strava

Connecte ton compte Strava pour importer automatiquement tes activités et alimenter tes stats personnelles.

⚙️ L'intégration Strava n'est pas configurée sur ce serveur.

Configuration admin (à faire 1 fois) :
  1. Crée une app sur strava.com/settings/api
  2. Configure le Authorization Callback Domain : localhost (dev) ou ton domaine
  3. Copie Client ID et Client Secret dans le fichier .env du serveur : 
    STRAVA_CLIENT_ID=12345
STRAVA_CLIENT_SECRET=abc123...
STRAVA_REDIRECT_URI=http://localhost:3000/api/strava/callback
  4. Redémarre le serveur (npm start) → cette section affichera le bouton "Connecter mon Strava".

Une fois configuré, tous les membres (admin compris) pourront relier leur compte Strava en un clic depuis cette page.

Ton compte Strava n'est pas encore lié. Clique pour voir les bénéfices et les permissions avant d'autoriser le Club.
